# No hedging copy

State the number and what it is. No "roughly", no "it seems", no apology for a real figure — a dashboard that hedges teaches the reader to distrust it.

**Why it exists.** Honesty belongs in the label, not in a softening adverb. "Roughly 14M rates" and "14M rates (payer-attested, not negotiated averages)" are both honest; only the second one is useful. Hedging language moves the uncertainty out of the data model and into the prose, where nobody can act on it.

**How to apply.** If a figure needs a caveat, put the caveat in the label or the tooltip where it can be precise. If a figure is an estimate, mark it as one. Then let the number stand.
