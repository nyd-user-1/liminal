# Reuse the kit

44 primitives and roughly 30 feature components already exist. Compose them; a genuinely new primitive is a deliberate, announced act — never an accident of local convenience.

**Why it exists.** Ten terminals building in parallel will each invent their own card, their own toolbar, their own empty state. Six months of that and the product reads like six products. The kit is the shared vocabulary that keeps one hand visible across every surface.

**How to apply.** Browse the live catalog at `/design-system` — every card there carries a click-to-copy import line, and the page's "Copy manifest" button emits the full inventory. If nothing fits, compose two things that do. If composition genuinely cannot express the pattern, add the primitive, say so explicitly in your report, and update the catalog in the same tranche so the next builder inherits it.
