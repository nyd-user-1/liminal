# Verified means exercised

"Done" means the surface was rendered and looked at — in both themes, at the widths that matter — not that the types happened to pass.

**Why it exists.** A green build proves the code compiles. It says nothing about whether the table overflows, whether the empty state is blank, or whether the dark theme turned the text the same colour as its background. Every one of those has shipped behind a passing type check.

**How to apply.** Sign in for real with a cookie, drive the page headless, and look at the pixels. Screenshot the surface you claim to have fixed. When you report, say what you rendered and at what width — an exit code is not evidence.
