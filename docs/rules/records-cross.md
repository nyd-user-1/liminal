# Records cross with one motion

A value that lives in another table wears the dotted-teal underline that wipes to solid on hover — one meaning, on every surface.

**Why it exists.** In a system this relational, the most common question a reader has is "can I go there?" Answering it with a different treatment on each page turns a navigational fact into a guessing game. One motion, one meaning: dotted teal means this value is a record elsewhere.

**How to apply.** Use the `RelatedLink` primitive from the kit. Plain text stays plain; a cross-record value gets the underline. Never hand-roll the treatment with a bare anchor and a border-bottom.
